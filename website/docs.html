Documentation

TestGen is an AI-powered CLI tool that automatically generates production-ready unit tests for your code. It supports JavaScript, TypeScript, Python, Go, and Rust.

Installation

TestGen is available as a standalone binary for macOS, Windows, and Linux.

macOS

Install via curl:

# Apple Silicon
curl -L https://github.com/princepal9120/testgen-cli/releases/latest/download/testgen-darwin-arm64 -o testgen

# Intel
curl -L https://github.com/princepal9120/testgen-cli/releases/latest/download/testgen-darwin-amd64 -o testgen

chmod +x testgen
sudo mv testgen /usr/local/bin/

Windows

Download the executable via PowerShell:

Invoke-WebRequest -Uri "https://github.com/princepal9120/testgen-cli/releases/latest/download/testgen-windows-amd64.exe" -OutFile "testgen.exe"

# Move to a folder in your PATH, e.g. C:\tools
move testgen.exe C:\tools\

Linux

Install via curl:

curl -L https://github.com/princepal9120/testgen-cli/releases/latest/download/testgen-linux-amd64 -o testgen
chmod +x testgen
sudo mv testgen /usr/local/bin/

Configuration

You need an API key from Anthropic (recommended) or OpenAI to use TestGen.

# Add to your ~/.zshrc or ~/.bashrc
export ANTHROPIC_API_KEY="sk-ant-..."

# Or for OpenAI
export OPENAI_API_KEY="sk-..."

Generating Tests

TestGen provides several commands for generating tests. Here are the most common use cases:

Single File

Generate tests for a single source file:

testgen generate --file=./src/utils.py --type=unit

Entire Directory

Process all files in a directory recursively:

testgen generate --path=./src --recursive --type=unit,edge-cases

Dry Run (Preview)

Preview what tests would be generated without writing files:

testgen generate --path=./src --dry-run --output-format=json

With Validation

Generate tests and validate they compile/run:

testgen generate --path=./src --validate --parallel=4

Cost Estimation

Estimate the API cost before generating:

testgen analyze --path=./src --cost-estimate

# Output:
# === Codebase Analysis ===
# Total files:     24
# Total lines:     3,456
# Est. functions:  89
#
# Estimated tokens: 45,000
# Estimated cost:   $0.38 USD

Supported Languages

TestGen supports multiple programming languages and their popular testing frameworks:

JavaScript / TypeScript

  • Frameworks: Jest, Vitest, Mocha
  • Extensions: .js, .jsx, .ts, .tsx

Python

  • Frameworks: pytest, unittest
  • Extensions: .py

Go

  • Frameworks: testing (standard library), testify
  • Extensions: .go

Rust

  • Frameworks: cargo test (built-in)
  • Extensions: .rs
