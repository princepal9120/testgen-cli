Installation

Get started in 3 simple steps. Works on macOS, Windows, and Linux.

1

Download the Binary

Get the latest release for your Mac

# Apple Silicon (M1/M2/M3)
curl -L https://github.com/princepal9120/testgen-cli/releases/latest/download/testgen-darwin-arm64 -o testgen

# Intel Mac
curl -L https://github.com/princepal9120/testgen-cli/releases/latest/download/testgen-darwin-amd64 -o testgen
2

Make it Executable & Move to PATH

Add testgen to your system commands

chmod +x testgen
sudo mv testgen /usr/local/bin/
3

Set Your API Key

Add to your ~/.zshrc or ~/.bashrc

# Anthropic Claude (recommended)
export ANTHROPIC_API_KEY="sk-ant-..."

# Or OpenAI
export OPENAI_API_KEY="sk-..."

testgen --version

Usage Examples

Single File

testgen generate --file=./src/utils.py --type=unit

Entire Directory

testgen generate --path=./src --recursive --type=unit,edge-cases

Dry Run (Preview)

testgen generate --path=./src --dry-run --output-format=json

With Validation

testgen generate --path=./src --validate --parallel=4

Cost Estimation

testgen analyze --path=./src --cost-estimate

=== Codebase Analysis ===
Total files:     24
Total lines:     3,456
Est. functions:  89

Estimated tokens: 45,000
Estimated cost:   $0.38 USD

Per-File Breakdown

testgen analyze --path=./src --detail=per-file --output-format=json

Check Coverage

testgen validate --path=./src --min-coverage=80

CI Pipeline

testgen validate --path=./src --fail-on-missing-tests --quiet
